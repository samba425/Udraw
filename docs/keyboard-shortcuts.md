# Keyboard shortcuts

`mod` = Ctrl on Windows/Linux, Cmd on macOS. Shortcuts are ignored while typing
in inputs/textareas or editing shape text. Plugin shortcuts are checked first.

## Tools (single key)

| Key | Tool |
| --- | --- |
| `V` | Select |
| `H` | Pan (hand) |
| `R` | Rectangle |
| `O` | Ellipse |
| `D` | Diamond |
| `T` | Text |
| `C` | Connector |
| `N` | Sticky note |
| `P` | Pen (freehand) |
| `E` | Eraser |

## Editing

| Shortcut | Action |
| --- | --- |
| `mod+Z` | Undo |
| `mod+Shift+Z` / `mod+Y` | Redo |
| `mod+A` | Select all |
| `mod+C` | Copy |
| `mod+X` | Cut |
| `mod+V` | Paste (with offset) |
| `mod+D` | Duplicate |
| `mod+G` | Group |
| `mod+Shift+G` | Ungroup |
| `Delete` / `Backspace` | Delete selection |
| `Escape` | Clear selection / close search / reset to Select tool |
| Arrow keys | Nudge selection by 1px (`Shift` = 10px) |

## File

| Shortcut | Action |
| --- | --- |
| `mod+S` | Save as JSON |
| `mod+P` | Export as PDF |
| `mod+F` | Find in diagram |

## View / mouse

| Input | Action |
| --- | --- |
| `Shift+1` | Fit diagram to screen |
| Mouse wheel | Zoom toward the cursor |
| Space / middle-drag | Pan |
| `Shift`+click | Multi-select |

> Note: the sparkle button in the toolbar opens the AI generator dialog; inside
> it, `mod+Enter` generates and `Escape` closes.
